# Button

Button component is used to create interactive and clickable elements within a user interface. Its primary functionality revolves around providing a standardized and reusable way to trigger actions or events when users interact with the application.

## Import

```js copy
import Button from "@repo/ui/button";
```

## Base usage

<ButtonBaseUsage />

  Open
