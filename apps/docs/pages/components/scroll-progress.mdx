# ScrollProgress

Displays a visual representation of the scroll progress on the page. It updates the progress bar as the user scrolls through the content.

Please, Scroll the page to see the result at the top of a current page.

## Import

```js copy
import ScrollProgress from "@repo/ui/scroll-progress";
```

## Base usage

```js copy
    <ScrollProgress />
```
